## 特色功能

可同时启动多个浏览器、多标签页同步执行提交操作

目前支持Chrome和Edge浏览器，由于华为商城登录可能会需要验证码，需要先执行登录操作

这里默认设置了浏览器的启动端口方便后续的操作可以顺利接管

辅助下单，实际抢购跟账号关联比较大，全靠运气
